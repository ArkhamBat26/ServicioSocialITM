Sistema de Reconocimiento Facial 
Software para la deteccion y reconocimiento de rostros para el Sistema de Gestion Estrategica

Empezando
Estas instrucciones le permitirán obtener una copia del proyecto en funcionamiento en su máquina 
local para fines de desarrollo y prueba. Consulte la implementación para obtener notas sobre cómo 
implementar el proyecto en un sistema en vivo.

Prerrequisitos
Computadora con camara web
Python
Para la instalación de Python, se requiere unicamente descargar el instalador de su pagina web 
oficial y seguir las instrucciones para su instalación.
Pip
Por defecto, con la instalación de Python se instalara easy_install, para poder gestionar los 
paquetes que requiera Python. Es recomendable instalar tambien pip. Pip es un sistema de gestión 
de paquetes utilizado para instalar y administrar paquetes de software escritos en Python. Para 
realizar la instalación de pip, tendremos que teclear la siguiente linea de instruccion:
easy_install pip
OpenCV
OpenCV es necesario para el manejo y procesamiento digital de imagenes. Para instalarlo hay que 
ejecutar la siguiente linea de codigo
pip install opencv-python
De igual manera, necesitaremos algunos modulos extra para el funcionamiento del reconocedor facil.
Estas librerias extra se instalaran con la siguiente linea de codigo.
pip install opencv-contrib-python 
Numpy
Para la instalación de numpy, se abrira la terminal de comandos y se procedera a teclear la siguiente 
linea
pip install numpy
Django
Para la instalación de Django, de igual manera utilizaremos la siguiente linea de ejecución
pip install django
